### My personal portfolio website!

Click [Here](https://chrisnng.github.io/Ng/) to visit it.

This site is created with create-react-apt and react-bootstrap.

### Stack

A single-paged portfolio website that focuses on the user interface. Emails are served to me through Nodemailer, Express, and Google's OAuth 2.0  on form submission. See [here](https://github.com/ChrisnNg/Ng-Backend) for the backend portion. Techstack: CSS, ReactJS, Express, and hosted on Heroku and Github pages
