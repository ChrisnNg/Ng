### Stack

This Site is created with [Facebook's create-react-app](https://github.com/facebook/create-react-app) and animations are primarily come from [react-reveal](https://www.react-reveal.com/). This repo is purely front-end. See [here](https://github.com/ChrisnNg/Ng-Backend) for the backend portion. This project is hosted on Github Pages. The contact me form utilizes nodemailer to send emails to me on the form submission.
